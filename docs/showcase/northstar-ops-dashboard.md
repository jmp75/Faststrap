# Northstar Ops Dashboard

`showcase/northstar_ops_dashboard.py` is the current flagship dashboard reference for Faststrap.

It exists to prove the heavier product side of the framework: dense information design, filters, export flows, and data-oriented UI surfaces.

## What It Demonstrates

- `DashboardLayout`
- `DashboardGrid`
- `MetricCard`
- `TrendCard`
- `KPICard`
- `Chart`
- `DataTable`
- `FilterBar`
- `DateRangePicker`
- `MultiSelect`
- `RangeSlider`
- `ExportButton`
- `NotificationCenter`
- `ThemeToggle`
