# Atlas Command Center

`showcase/atlas_command_center.py` is a v1 dashboard reference that demonstrates the new Faststrap component wave in a dense operations product surface.

## Highlights

- `CommandPalette` and `CommandItem` for command-center navigation
- `ChartJS` optional integration for richer charting
- `ModernToast` and `ModernToastStack` for polished feedback
- `Timeline` and `TimelineItem` for incident activity
- `AvatarGroup`, `StatusBadge`, and `BadgeGroup` for team/status context
- `DataTable.query_params()` and `DataTable.page_url()` for reusable table state
- HTMX-ready `Pagination`
- `ConfirmAction` for safe destructive actions
- `theme_variant_css()` for light/dark surface variants

## Run Locally

```bash
python showcase/atlas_command_center.py
```

The app serves on port `5028`.
